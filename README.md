# ⛈️ CloudBurst India — Weather Website

## ✅ QUICKSTART (2 steps)

### Step 1 — Open terminal in this folder
```
cd weather-app
```

### Step 2 — Run the server (No install needed — uses built-in Node.js)
```
node server.js
```

### Step 3 — Open browser
```
http://localhost:3000
```

---

## 🌐 Deploy to Internet (Free)

### Option A — Vercel (Recommended, 1 click)
1. Go to vercel.com → New Project
2. Drop this folder OR connect GitHub repo
3. Done! Your site is live at `https://yourapp.vercel.app`

### Option B — Netlify
1. Go to netlify.com → Drop `index.html` in deploy zone
2. Live immediately at `https://random-name.netlify.app`

### Option C — cPanel / Shared Hosting
1. Upload `index.html` to `public_html/` folder
2. Visit your domain — it works!

---

## ⚠️ Why `file://` doesn't work
Browsers block API calls from local files (CORS security policy).
Always use `node server.js` for local development.
